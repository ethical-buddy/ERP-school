# API Phase Plan (Deferred)

Current delivery scope is web application only.

Planned phase for API:

1. Add Django REST Framework and JWT auth.
2. Expose school-scoped API endpoints for each module.
3. Add role-based permission classes by module.
4. Add pagination, filtering, and strict throttling.
5. Add audit logs for all write API endpoints.
6. Add mobile-optimized endpoints for student and staff apps.
7. Add contract tests and OpenAPI docs.
